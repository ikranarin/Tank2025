# Tank2025 🚗💥

Tank2025 is a **JavaFX-based 2D tank battle game** where the player controls a tank, navigates through a maze-like map, and fights enemy tanks.

The game includes enemy AI, shooting mechanics, score tracking, player lives, explosion effects, and a start/pause system.

---

## 🎮 Gameplay

The player controls a yellow tank and must survive while destroying enemy tanks.

Enemy tanks move randomly around the map and shoot bullets toward the player. The player gains points by destroying enemy tanks while avoiding enemy bullets and collisions.

The game ends when the player loses all lives.

---

## 🕹 Controls

| Key | Action |
|----|----|
| ↑ ↓ ← → | Move the tank |
| X | Shoot / Start the game |
| P | Pause the game |
| R | Restart the game |
| ESC | Exit the game |

---

## ⚙ Features

- JavaFX based 2D tank game
- Player tank movement and rotation
- Enemy tanks with simple AI
- Bullet shooting system
- Collision detection
- Player lives system
- Score tracking
- Tank explosion effects
- Enemy respawn system
- Start screen with animation
- Pause and restart functionality
- Tank animation while moving

---

## 🧠 Enemy Behavior

Enemy tanks:

- Move in random directions
- Change direction every few seconds
- Shoot bullets in the direction they are facing
- Respawn after being destroyed

---

## 🗺 Map System

The game uses a **grid-based tile system**.

- Tile size: `18px`
- Walls are generated using a map matrix
- Tanks and bullets check collisions with map tiles
